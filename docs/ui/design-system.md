# Design System

## Philosophy

- Clean
- Clear
- Non-generic
- Functional over decorative

---

## Core Principles

1. Clarity > Visual complexity
2. Consistency > Creativity
3. Reusable components only
4. Minimal cognitive load

---

## Components

- Button
- Card
- Input
- StudyCard
- DeckItem

---

## Layout

- Spacing system (8pt grid)
- Consistent padding
- Clear hierarchy

---

## Colors

- Primary
- Secondary
- Background
- Surface
- Error
- Success

---

## Typography

- Title
- Subtitle
- Body
- Caption

---

## States

Must exist for:
- Empty state
- Loading state
- Error state

---

## Study Screen Rules

- Focus on content
- No distractions
- Clear front/back separation
- Smooth transitions

---

## Forbidden

- Random styles per screen
- Inline styles everywhere
- Unstructured UI
